# sulinsardoschau.com

Personal academic website for Sulin Sardoschau — a minimal, hand-coded static site
(Home · Research · Policy). Hosted on GitHub Pages at **sulinsardoschau.com**.

## Editing & publishing
- Edit the `.html` files directly (text lives between tags). Preview locally with
  `python3 -m http.server` in this folder, then open the printed localhost URL.
- **Publishing is manual:** changes go live only when you `git commit` and `git push`.
  Nothing syncs automatically from Google Drive or anywhere else.
- Paper/CV PDFs are **links to Google Drive files** (shared "anyone with link"), not
  files in this repo. Keep only web-safe versions shared.

## Structure
- `index.html` — home / bio
- `research.html` — working papers, publications, work in progress
- `policy.html` — policy & public writing
- `style.css` — all styling
- `assets/` — photo + favicon
- `CNAME` — custom domain (do not delete)
